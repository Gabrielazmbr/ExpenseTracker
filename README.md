# Expense Tracker

## Description

Expense Tracker is a Python-based application that helps users manage their expenses efficiently. The program allows users to:

Add expenses with categorization.

## View expense history.

Get a summarized report of their expenses.

Generate visual charts based on recorded expenses.

## Features

Add an Expense: Input and categorize expenses easily.

View Expense History: Keep track of past expenses.

Expense Summary: Get a breakdown of expenses by category and total spending.

Data Visualization: Generate charts to visualize expense trends.

## Installation

To use Expense Tracker, follow these steps:

Clone the repository:

git clone https://github.com/Gabrielazmbr/ExpenseTracker.git

Navigate to the project directory:

cd EXPENSE_TRACKER

Install dependencies (if applicable):

pip install -r requirements.txt

Run the application:

python app.py

## Usage

When you run the program, you will see the following menu:

Please select an option from the menu:
1. Add an expense
2. See my expense history
3. View expenses summary
4. Exit!

Select 1 to add a new expenses, you will be asked to add the amount of the expense, the date, the category (can create new categories, or use categories created previusly.

Select 2 to view past expenses, in a table that tracks and totalize them.

Select 3 to see a summarized report with charts, the chart is automatically downlowded containing relevant stadistical information of the expenses.

Select 4 to exit the program.
